## Important Note
The site is not currently hosted and **is still under development**.

## Description
For get book covers I use "bookvoed" (https://www.bookvoed.ru). Users can:

- Create an account
- Log in
- Log out
- Add reviews
- Edit reviews
- Delete reviews
- View other users' reviews
- Browse books
- Browse authors
- Users can have admin rights
